## 📌 Project Overview
This project showcases an **interactive Power BI dashboard** built to analyze **financial transaction performance and user onboarding behavior in a fintech environment**.

The dashboards help track **transaction success rates, payment method usage, and the user onboarding funnel**, providing insights into how users move from **app download to full KYC completion**.

---

## 📈 Dashboard 1 – Financial Transaction Performance
This dashboard focuses on monitoring transaction activity and payment performance.

**Key Metrics**
- Transaction Volume  
- Transaction Success Rate  
- Payment Mode Distribution  
- Transaction Failure Trends  

**Objective**
- Track transaction performance  
- Identify transaction failure patterns  
- Analyze payment method usage  

---

## 👤 Dashboard 2 – User Onboarding & KYC Analytics
This dashboard analyzes the **user onboarding journey and KYC verification progress**.

**Key Metrics**
- App Downloads  
- User Activation  
- Wallet Activation Attempts  
- Full KYC Completion  

**Objective**
- Track onboarding performance  
- Monitor KYC verification progress  
- Identify drop-offs in the user conversion funnel
